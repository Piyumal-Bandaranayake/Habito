# Habito - Habit & Mood Tracker

Habito is a modern, intuitive Android application designed to help you build lasting habits and monitor your emotional well-being. By combining habit tracking with mood logs, Habito provides a holistic view of your daily life and personal growth.

![Habito App](https://via.placeholder.com/800x400.png?text=Habito+App+Preview)

## ‚ú® Features

- **Habit Tracking**: Set daily goals, check off completed habits, and maintain streaks to stay motivated.
- **Mood Monitoring**: Log your daily mood and see how your habits influence your emotional state.
- **Progress Analytics**: Detailed charts and statistics to visualize your improvement over weeks and months.
- **Seamless Onboarding**: A beautiful introductory guide to help you get started in seconds.
- **Customizable Experience**: Manage your preferences through a clean and accessible Settings interface.

### Prerequisites

- Android Studio Flamingo or newer
- JDK 17
- Android SDK Level 33+

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Piyumal-Bandaranayake/Habito.git
   ```
2. **Open in Android Studio**:
   File > Open > Select the `Habito` folder.
3. **Sync Project**:
   Ensure Gradle sync completes successfully.
4. **Run**:
   Select an emulator or physical device and click the **Run** button.
